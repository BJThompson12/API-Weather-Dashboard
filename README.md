# API-Weather-Dashboard
## Description
A weather dashboard using HTML, CSS, Javascript and API's to update the weather. 

## Installation

N/A

## Technologies 

HTML, JavaScript, CSS, Tailwind and API calls. 

## Usage

The dashboard is to be used to get the current and 5 day forecast for selected cities.
